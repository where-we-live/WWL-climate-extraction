# climate-metrics<br>
This repository is has been used for a number of steps as part of the PDI process. 

Currently, there are two core files that are doing all the heavy lifting.

1. /src/climate_metrics.R
2. PDI.Rmd - file that calls climate_metrics.R

Running the Rmd file generates a number of plots in /plots and output csv files in /outputs

The following folders contain:<br>
/src - all code and functions<br>
/data - a number of folders are withing this location, including climate extracted data as well as survey data used as part of this analysis<br>
/geodata - the geographic shape files needed to extract data<br>
/plots - all plots/graphs generated from the code<br>
/outputs - file outputs that are csvs<br>
